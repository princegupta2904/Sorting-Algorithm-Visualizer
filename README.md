# Sorting Algorithm Visualizer

An interactive web-based Sorting Algorithm Visualizer built using HTML, CSS, and JavaScript.  
This project demonstrates Bubble Sort through real-time animations to help understand comparisons, swaps, and time complexity.

## Features
- Bubble Sort visualization
- Adjustable array size
- Speed control
- Real-time animations
- Clean and simple UI

- HTML

- JavaScript
